# TIPS©

**Experiment for Many Ways to Bind Paper / 종이를 엮는 수많은 방법을 실험하다**

TIPS 브랜드 포트폴리오 웹 — 학습용 웹디자인 프로젝트.
정적 사이트(HTML/CSS/JS)로, GitHub Pages에서 바로 동작합니다.

## 로컬 실행
```bash
python3 -m http.server 5500   # http://127.0.0.1:5500
```

## 구조
- `index.html` — 메인(마크업 + 인터랙션 스크립트)
- `css/site.css` — 스타일
- `assets/` — 이미지(`images/tips/`)·폰트·아이콘
- `vendor/` — 라이브러리(jQuery, GSAP, SplitType, Masonry, imagesLoaded) + Webflow 런타임

> 참고: 본 사이트는 reallyupthere.com의 구조/인터랙션을 학습 목적으로 재구성한 뒤 TIPS 브랜드로 리브랜딩한 **비상업 스터디 프로젝트**입니다. 일부 폰트·라이브러리·레퍼런스 이미지는 각 원저작자에게 권리가 있습니다.
